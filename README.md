# Desafio Relógio API

API REST desenvolvida em Java com Spring Boot para gerenciamento de relógios.

## Tecnologias utilizadas
- Java 21
- Spring Boot
- Spring Data JPA
- Bean Validation
- PostgreSQL
- Maven

## Funcionalidades
- Cadastro de relógios
- Atualização de dados
- Listagem paginada
- Ordenação personalizada
- Validações de entrada
- Uso de DTOs e Mappers

## Arquitetura
- Controller
- Service
- Repository
- DTO
- Mapper
- Entidades com UUID
- Tratamento de exceções

## Objetivo do projeto
Projeto desenvolvido com foco em estudo de:
- Arquitetura de APIs REST
- Boas práticas com Spring Boot
- Separação de responsabilidades
- Padrões utilizados no mercado

## Como executar
1. Clone o repositório
2. Configure o banco de dados no `application.properties`
3. Execute a aplicação pela classe principal
